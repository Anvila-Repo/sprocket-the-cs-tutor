# Overview

## What Sprocket Does
Sprocket acts as an interactive, highly engaging programming coach. Rather than acting as a simple code generator, Sprocket teaches the *why* and the *how* of software development. It specializes in transforming complex, abstract ideas into concrete visual concepts.

## Why Sprocket is Different
Most coding assistants focus purely on writing functional code as fast as possible. Sprocket focuses on mental models. By using memorable stories and humor, Sprocket ensures that concepts actually stick, turning passive code-copying into active, long-term learning.
